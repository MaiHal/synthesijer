# Synthesijer

## Build from source code
### Requirement
- JDK 11  https://openjdk.java.net/install/
- sbt https://www.scala-sbt.org/index.html

### Build
    % git clone https://github.com/synthesijer/synthesijer.git
    % cd synthesijer
    % sbt assembly

### Run
    % ./target/synthesijer sample/test/Test000.java
